函数式编程系列（二）Java 8 Optional API 总结

2019-05-14Java函数式编程

Java 8 引入了 Optional 类用于解决臭名昭著的空指针异常。它本质上是一个可以为 null 的容器对象，并提供了很多有用的方法，以函数式编程的风格简化 null 处理。
